# 🛒 SellerPro — E-Commerce Seller Dashboard

A responsive seller dashboard for managing products, orders, reviews, and customers — built with plain **HTML, CSS, and vanilla JavaScript**. No frameworks, no build step.

## Features

- **Dashboard** — revenue/orders/products stats, revenue chart, recent orders, notifications
- **Products** — searchable/filterable grid, category tabs, quick-view modal, add-to-cart
- **Orders** — filterable order table with status timeline modal
- **Reviews** — review stats and searchable customer feedback
- **Customers** — customer directory with VIP/Active/New badges
- **Cart** — quantity controls, promo code, checkout flow
- Fully responsive (collapsible sidebar on mobile)
